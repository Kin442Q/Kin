### Как создать нового учителя через админ-панель?

1. **Войдите как администратор:**
   ```
   Вкладка: Администратор
   Email: admin@kindergarten.tj
   Пароль: Admin123456!
   ```

2. **Перейдите в раздел «Учителя»** в боковом меню

3. **Нажмите кнопку «Зарегистрировать учителя»**

4. **Заполните форму:**
   - **ФИО** (обязательное) — полное имя учителя
   - **Email** (необязательное) — генерируется автоматически из номера
   - **Пароль** (обязательное) — начальный пароль для входа
   - **Группа** (обязательное) — выберите группу из списка
   - **Номер телефона** (обязательное) — используется для входа в систему

5. **Нажмите «Создать»**

6. **Сообщите учителю его номер и пароль:**
   ```
   Номер телефона: +992 90 123 45 67
   Пароль: YourPassword123!
   
   Вход: на сайте выберите вкладку "Воспитатель"
   ```

## 📋 Как дать логины и пароли учителям?

### Способ 1: Распечатать или отправить по email
Скопируйте эту таблицу и отправьте каждому учителю их данные:

```
Email: teacher1@kindergarten.tj
Пароль: Teacher123456!

URL для входа: http://localhost:5174 (или ваш доменный адрес)
```

### Способ 2: Создать документ
Создайте общий документ со всеми логинами и распечатайте:
- Email для входа
- Начальный пароль
- Ссылка на систему

### Способ 3: Управление через админ-панель (будущее)
В будущем добавится функция для сброса пароля прямо из админ-панели.

⚠️ **ВАЖНО:** Учителя должны изменить пароль при первом входе на более безопасный!

## 🔄 Как переоздать учетные данные?

Если вам нужно переоздать данные и учетные записи, выполните:

```bash
cd server
npm run seed
```

Это переустановит всех пользователей, группы, студентов и демо-данные.

## 📝 Примечания
- Демо-пресеты были **удалены** из интерфейса входа для безопасности
- Все пароли **зеша**нированы в БД с использованием bcryptjs
- Система производит логирование всех попыток входа
