## 📋 Как дать логины и пароли учителям?

### Способ 1: Распечатать или отправить по email
Скопируйте эту таблицу и отправьте каждому учителю их данные:

```
Email: teacher1@kindergarten.tj
Пароль: Teacher123456!

URL для входа: http://localhost:5174 (или ваш доменный адрес)
```

### Способ 2: Создать документ
Создайте общий документ со всеми логинами и распечатайте:
- Email для входа
- Начальный пароль
- Ссылка на систему

### Способ 3: Управление через админ-панель (будущее)
В будущем добавится функция для сброса пароля прямо из админ-панели.

⚠️ **ВАЖНО:** Учителя должны изменить пароль при первом входе на более безопасный!

## 🔄 Как переоздать учетные данные?

Если вам нужно переоздать данные и учетные записи, выполните:

```bash
cd server
npm run seed
```

Это переустановит всех пользователей, группы, студентов и демо-данные.
